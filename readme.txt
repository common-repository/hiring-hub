== Description ==

**Create a job portal and career page with WordPress**. Manage job boards and run recruitment activities, all within your WordPress website. Define job specifications, apply link, and compose the layout using Site Editor.

#### Plugin features

* **Highly customizable job specifications** - Define the job characteristics and specifications like workplace ("remote", "hybrid", etc.), employment type ("full time", "part time", etc.) and other, without the limits. Add as many or as little specifications as you need.
* **Add "apply on" link to the job posts** - Define the link under which candidates can apply to the job listing, in the panel while editing the job post.
* **Built specifically for the Block Editor & Full Site Editing** - Take advantage of the most recent features available in WordPress: this plugin works perfectly with the block themes.
* **A bunch of custom blocks & block patterns** - Construct the site in the way you need it, with the WYSIWYG experience.
* **Query loop filters** - Build rich job post search/filter sections. Thanks to this, candidates will be able to filter-out the job listings and find the jobs theyâ€™re interested about.
* **<a href="https://schema.org/JobPosting">JobPosting schema.org</a> support** - Job specifications can be mapped to the schema.org properties, powering the rich snippets in search results. 25 properties is defined by default, available to use out of the box!
* **Add, edit, delete job listings** using the custom "job" post type.

#### "Careers" theme

Check the "Careers" theme for WordPress, built specifically for the Hiring Hub plugin. This theme showcase features of the Hiring Hub plugin, making it easier to get started - you can create the Hiring/Careers website in minutes.

Careers theme is compatible with Full-site Editing, and is available on WordPress.org for free: <a href="https://wordpress.org/themes/careers/">https://wordpress.org/themes/careers/</a>
